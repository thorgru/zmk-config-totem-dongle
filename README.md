Flash Dongle w/ settings reset
Flash Left w/ settings reset (power On)
Flash Right w/ settings reset (power On)
Flash Right with firmware
Press Right Reset Button
Wait 20 seconds
Flash Left with firmware
Press Left Reset Button
Wait 20 seconds
